# Simple C Calculator

A basic calculator written in C that supports addition, subtraction, multiplication, and division.

## Features

- Integer-based input
- Input loop
- Exit command (`exit`)
- Error handling for division by zero

## Why?

Built for learning purposes while studying C as part of my journey into software development and DevOps.

I am more familiar with Python, but this journey has just started and i'm hungry to learn more about coding!
